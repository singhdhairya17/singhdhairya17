## `$ cat /projects/nexus960`

> **A Chess960 platform built for people who actually care about chess.**

| Layer | Stack | Purpose |
|---|---|---|
| **Frontend** | Next.js · Vercel | Real-time game UI |
| **Backend** | Fastify · Socket.io · Railway | Game engine & WebSocket server |
| **Database** | Supabase PostgreSQL · PgBouncer | Player data, game history |
| **Cache** | Upstash Redis · BullMQ | Move queues, session state |
| **AI** | Stockfish WASM (client-side) | Engine analysis without server cost |
| **Rating** | NX Rating (Glicko-2 variant) | Custom ELO built from scratch |

```
Features shipping:
  ✦  Real-time multiplayer · Chess960 position randomizer
  ✦  Aria — AI training companion
  ✦  N1 — custom neural engine (ResNet · 10–15M params · RTX 4050)
  ✦  Style analytics · Tournament system
  ✦  NX Rating: Glicko-2 based · uncertainty-aware
```
